# 极简待办 — 本地化极简办公工具

> 你的待办事项，只属于你自己。

一款纯本地、零依赖云服务的 Windows 桌面待办应用。数据留在你的电脑上，界面融入你的桌面中——安静、专注、不打扰。

---

## 为什么选择极简待办？

| 传统待办应用 | 极简待办 |
|:---:|:---:|
| 需要注册账号 | 打开即用 |
| 数据上传云端 | 数据存储在本地 |
| 广告推送通知 | 零广告、零推送 |
| 界面复杂臃肿 | 极简透明，融入桌面 |
| 依赖网络连接 | 完全离线运行 |

---

## 核心理念

### 🏠 本地优先
所有数据以 JSON 格式存储在本地 `%AppData%\DesktopToDo\` 目录下，不依赖任何云服务、不联网、不上传。你的待办事项只属于你自己。

### 🪶 极简设计
- **透明窗口** — 半透明界面融入桌面，不遮挡工作内容
- **嵌入桌面** — 可开启壁纸模式，待办列表直接嵌入桌面图标层
- **轻量运行** — 基于 .NET 6.0 WPF，资源占用极低
- **系统托盘** — 最小化后隐藏到托盘，随时唤出

### 📋 办公实用
- **任务优先级** — 低 / 普通 / 高 / 紧急，颜色一目了然
- **重复提醒** — 支持每天、每周、每月、每季度、每年、工作日循环
- **次级清单** — 每个任务可拆分子步骤，点击展开/折叠
- **提前提醒** — 可配置提前提醒时间，不错过任何截止日期

---

## 功能一览

### 任务管理
- 添加、编辑、完成、删除任务
- 四级优先级（低 / 普通 / 高 / 紧急）
- 次级清单：回车分段输入，点击主任务展开/折叠
- 已完成的次级任务自动标记，编辑时保留完成状态

### 提醒系统
- 自定义提醒日期和时间
- 可配置提前提醒分钟数
- 重复任务自动生成下一周期

### 重复规则
- 不重复 / 每天 / 每周 / 每月 / 每季度 / 每年 / 工作日
- 支持自定义结束条件（永不结束 / 指定次数 / 指定日期）

### 个性化设置
- 窗口透明度调节
- 字体和字号自定义
- 四种主题色（淡绿 / 白色 / 灰色 / 自定义）
- 窗口位置和大小自动记忆
- 开机自启动

### 数据安全
- 本地 JSON 存储，不上传任何数据
- 支持导出备份 / 导入恢复
- 一键清空已完成任务

## 快速开始

### 环境要求

- Windows 7 / 10 / 11
- [.NET 6.0 SDK](https://dotnet.microsoft.com/download/dotnet/6.0)

### 构建运行

```bash
# 克隆项目
git clone <repository-url>
cd DesktopToDo

# 编译
dotnet build

# 运行
dotnet run
```

### 发布为独立可执行文件

```bash
dotnet publish -c Release -r win-x64 --self-contained true
```

发布产物位于 `bin\Release\net6.0-windows\win-x64\publish\`，可直接分发使用。

## 使用指南

### 基本操作

| 操作 | 说明 |
|:---|:---|
| **添加任务** | 点击 `➕ 添加` 按钮 |
| **编辑任务** | 点击 `✏️` 按钮 |
| **完成任务** | 点击 `✓` 按钮（重复任务自动生成下一期） |
| **删除任务** | 点击 `🗑️` 按钮 |
| **移动窗口** | 拖动窗口任意位置 |
| **最小化** | 点击 `—` 按钮隐藏到系统托盘 |

### 次级清单

添加任务时，在输入框中：
- **第一行** → 主任务内容
- **回车后的每一行** → 自动成为次级清单项

```
项目周报整理
收集各部门数据
撰写总结文档
提交审核
```

点击主任务或左侧 `▶` 按钮可展开/折叠次级清单。

### 提醒设置

1. 勾选「设置提醒」
2. 选择提醒日期和时间
3. 在设置中调整提前提醒分钟数
